# 📖 About

**CYBORG 2077** is a futuristic single-page web experience inspired by cyberpunk aesthetics.

The project focuses on **parallax scrolling**, where multiple background and foreground elements move at different speeds while scrolling, creating depth and making the website more engaging. It combines modern UI design with smooth animations to deliver an interactive browsing experience.

This project was created as part of a **Techfest Web Development Challenge**.

---

# ✨ Features

- 🌌 Multi-layer Parallax Scrolling
- 🤖 Cyberpunk-inspired User Interface
- ⚡ Smooth Scroll Animations
- 💎 Glassmorphism Design
- 🎨 Neon Glow Effects
- 🖱️ Interactive Cursor Effects
- 📊 Scroll Progress Indicator
- 🚀 Animated Hero Section
